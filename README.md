**Tech Stack:** Flutter, Dart, Firebase Authentication, Cloud Firestore

**Description:**  
A cross-platform mobile Notes application built with Flutter and Firebase. Users can sign up/login and create, view, edit, or delete notes. The app features a clean, responsive UI and uses Firestore for real-time data storage.

**Features:**  
- User Authentication (Firebase Auth)  
- Create, read, update, delete (CRUD) notes  
- Cloud Firestore backend integration  
- Responsive UI design  

**Setup Instructions:**  
1. Replace Firebase API keys in `lib/firebase_options.dart` with your own Firebase project credentials.  
2. Run `flutter pub get`  
3. Launch the app on your preferred device or emulator  

**Notes:**  
- Firebase API keys are removed for security.  
- This repo is meant for code review and demonstration purposes.
